# Secure Login System (Cyber Security Project)

## Description
This project demonstrates a secure login system with password hashing and authentication.

## Features
- User Registration
- Secure Login
- Password Hashing
- Session Management

## Technologies Used
- Python
- Flask
- SQLite

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/
